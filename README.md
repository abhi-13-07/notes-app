# Google Notes Clone

This is a **Google Notes Clone** with simple features, build out of **MERN stack**

### Folder Structure

- client - frontend (React App)
- server - backend (Node.js Server)

### How to run locally

**Client**

```
cd client/
npm install
  //&
npm start
```

**Server**

```
cd server/
npm install
  //&
npm run dev
```
